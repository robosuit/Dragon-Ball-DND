How To Save Your Character

  1. Fill in your character data on the sheet.
  2. Use New Slot once, then Save Slot for quick in-browser saves.
  3. Use Load Slot any time to switch characters.
  4. Optional backup: use Save to Device to download a JSON file.
  5. On a new device, open the app and use Load from Device.
